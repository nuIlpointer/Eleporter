# Eleporter

## これはなに

IronElevatorsに触発されて作った、マインクラフトのエレベータ的プラグインです。
特徴は以下の通りです。

・エレベータ化するブロックを複数指定可能

・移動上限範囲の設定可能

・上昇/下降それぞれで効果音設定可能

・軽い(初版で5.47KB)

## 使い方

1.コンフィグでエレベータとして使うブロックを追加する

2.追加したブロックの上でジャンプ/スニークで上昇/下降
